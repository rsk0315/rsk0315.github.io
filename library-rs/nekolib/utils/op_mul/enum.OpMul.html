    OpMulV,
    _Marker(T),

Expand description

積を返す演算を持つ。

std::ops::Mul により定義される。 単位元は One、逆元は MulRecip で定義する。 結合法則を満たすときは MulAssoc、交換法則を満たすときは MulComm を実装することで示す。
